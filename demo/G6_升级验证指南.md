## 🔍 常见问题排查

### 问题 1: 图谱不显示或白屏

**可能原因**:
- G6 版本未正确安装
- 容器尺寸问题

**解决方法**:
```bash
# 检查 G6 版本
npm list @antv/g6
# 应该显示: @antv/g6@4.8.24

# 强制重新安装
npm install @antv/g6@^4.8.24 --force
```

### 问题 2: TypeScript 类型错误

**可能原因**:
- IDE 缓存未更新
- @types 包冲突

**解决方法**:
```bash
# 清除 TypeScript 缓存（VS Code）
# Ctrl+Shift+P -> "TypeScript: Restart TS Server"

# 或者重启 IDE
```

### 问题 3: 节点或边操作失败

**症状**:
- 添加节点/边时报错
- 删除节点时报错
- 事件监听不生效

**排查步骤**:
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误信息
3. 检查调试日志：
   - `=== 关系视图数据调试 ===`（OntologyRelationshipView）
   - `=== 样式化后的数据 ===`（OntologyRelationshipView）

### 问题 4: 布局切换失败

**可能原因**:
- `changeData()` 方法调用问题
- 数据格式错误

**解决方法**:
检查控制台是否有 "Layout update failed" 错误，并查看具体错误信息。
