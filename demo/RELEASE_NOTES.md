## 📊 系统概述

攻击路径推演系统是一个基于React的现代化网络安全仿真平台，支持攻击路径分析、防御策略仿真、知识图谱可视化等核心功能。

**综合评分：** 9.0/10 ✅  
**E2E测试通过率：** 90.9% ✅  
**离线可用性：** 10/10 ✅

## 📦 安装部署

### 开发环境

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问系统
# http://localhost:3009
```

### 生产环境

```bash
# 1. 构建生产版本
npm run build

# 2. 部署dist目录到Web服务器
# 支持Nginx、Apache、IIS等

# 3. 配置Web服务器
# 确保支持SPA路由（所有路由指向index.html）
```
