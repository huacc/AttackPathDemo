# 2D网络沙盘推演可视化设计规范

> **文档版本**: v1.0
> **创建日期**: 2026-03-14
> **技术选型**: 2D方案（基于AntV G6），不使用3D

---

## 一、技术选型说明

### 1.1 为什么选择2D

- **开发成本低**：G6成熟稳定，API简单
- **调试成本低**：2D调试简单，无需处理复杂的3D坐标系
- **性能更好**：2D渲染性能优于3D，支持更多节点
- **用户认知成本低**：网络拓扑图用户更习惯2D视图

### 1.2 2D增强方案

- 使用**分层布局**模拟纵深感
- 使用**颜色深浅**表达层级关系
- 使用**动画效果**增强视觉冲击力
- 使用**区域背景**区分安全域

### 3.7 画布状态切换规则

**不同模式下画布的状态变化总结**：

| 模式 | 触发方式 | 画布变化 | 新增元素 | 消失元素 |
|-----|---------|---------|---------|---------|

| 防御部署 | 进入防御仿真 | 防御设备高亮，防御关系线显示 | 防御设备节点、防御关系边 | 攻击路径 |
| 防御仿真 | 点击开始仿真 | 攻击重放+防御响应动画 | 阻断标记、防御响应徽章 | 无 |

**状态切换动画**：
- 模式切换时，画布整体淡出(opacity 1→0, 300ms)再淡入(opacity 0→1, 300ms)
- 新增元素从透明淡入(opacity 0→1, 500ms)
- 消失元素淡出后移除(opacity 1→0, 300ms)

```typescript
function toggleLayer(layerName: string, visible: boolean) {
```

**时间轴播放实现**：
```typescript
class TimelinePlayer {
  private currentTime: number = 0;
  private speed: number = 1;
  private playing: boolean = false;
  private events: TimelineEvent[];

  play() {
    this.playing = true;
    this.tick();
  }

  pause() {
    this.playing = false;
  }

  setSpeed(speed: number) {
    this.speed = speed;
  }

  private tick() {
    if (!this.playing) return;

    this.currentTime += 0.1 * this.speed;

    // 触发当前时间点的事件
    const currentEvents = this.events.filter(
      e => Math.abs(e.time - this.currentTime) < 0.1
    );

    currentEvents.forEach(event => {
      this.executeEvent(event);
    });

    // 继续下一帧
    requestAnimationFrame(() => this.tick());
  }

  private executeEvent(event: TimelineEvent) {
    switch (event.type) {
      case 'attack':
        animateAttack(event.nodeId, event.action);
        break;
      case 'defense':
        animateDefenseResponse(event.nodeId, event.action);
        break;
      case 'alert':
        showAlert(event.data);
        break;
    }
  }
}
```

```typescript
function focusNode(nodeId: string) {
  const node = graph.findById(nodeId);
  const { x, y } = node.getModel();

  // 缩放到节点
  graph.zoomTo(1.5, { x, y });

  // 高亮节点及其邻居
  const neighbors = node.getNeighbors();
  graph.setItemState(node, 'focus', true);
  neighbors.forEach(neighbor => {
    graph.setItemState(neighbor, 'related', true);
  });

  // 淡化其他节点
  graph.getNodes().forEach(n => {
    if (n !== node && !neighbors.includes(n)) {
      graph.setItemState(n, 'dimmed', true);
    }
  });
}
```

```typescript
function highlightPath(pathNodes: string[]) {
  // 清除之前的高亮
  graph.getNodes().forEach(node => {
    graph.clearItemStates(node);
  });
  graph.getEdges().forEach(edge => {
    graph.clearItemStates(edge);
  });

  // 高亮路径节点
  pathNodes.forEach(nodeId => {
    graph.setItemState(nodeId, 'path-highlight', true);
  });

  // 高亮路径边
  for (let i = 0; i < pathNodes.length - 1; i++) {
    const edges = graph.findAllByState('edge', 'default');
    const edge = edges.find(e => {
      const model = e.getModel();
      return model.source === pathNodes[i] && model.target === pathNodes[i + 1];
    });
    if (edge) {
      graph.setItemState(edge, 'path-highlight', true);
    }
  }
}
```
